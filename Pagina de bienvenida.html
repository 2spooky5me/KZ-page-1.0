¿Que es Null?

NULL fue un proyecto encargado por el gobierno de Nueva Meca como una táctica para ganar la Guerra Cromag. tenía como objetivo producir súper soldados con poderes precognitivos utilizando una droga militar creada específicamente para el proyecto llamado Chronos.
